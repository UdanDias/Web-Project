BibliOS — Online Library Management System
==========================================
Group Assignment | Web Technologies

FILE STRUCTURE
--------------
index.html        Home page (hero, slider, stats, book cards)
books.html        Book Catalog (searchable table, modal popup)
membership.html   Membership Registration (form + JS validation)
account.html      My Account (tables, accordion, show/hide history)
contact.html      Contact Us (form, FAQ accordion, hours)
css/style.css     All styling (no Bootstrap needed — pure CSS)
js/script.js      All JavaScript + jQuery features
js/jquery.min.js  jQuery shim (CDN is loaded automatically)

ASSIGNMENT REQUIREMENTS MET
----------------------------
✅  5 Pages with meaningful titles
✅  Same navigation menu on every page
✅  External CSS file (css/style.css)
✅  JavaScript: live date, counter animation, validation
✅  jQuery: image slider, table filter, modal, accordion, slideToggle
✅  Form with JS validation (membership.html — 9 fields validated)
✅  Table (books.html — BK-001 to BK-012 with filter)
✅  JS Date display in navbar
✅  Responsive layout (CSS Grid + Flexbox, mobile breakpoints)
✅  Professional layout (Navy #1E3A5F / Gold #F8B400 scheme)

GROUP MEMBER RESPONSIBILITIES
------------------------------
1  → index.html (Home page content)
2  → books.html (Book Catalog page)
3  → membership.html (Registration form)
4  → account.html (My Account page)
5  → contact.html (Contact page)

8  → js/script.js — jQuery slider, accordion, filter

HOW TO RUN
----------
Open index.html in any modern browser. No server needed.
Internet connection required for Google Fonts + jQuery CDN.
